# GZ — Agent Detection & Response
### פרוטוטייפ דשבורד (Mockup)

קובץ HTML בודד, ללא תלויות התקנה — פותחים בדפדפן וזהו.

## מה זה
דמו ויזואלי של הדשבורד שמתואר במסמך המוצר של GZ: מעקב חי אחרי פעילות מסחר של AI agent, זיהוי חריגות, ו-Kill Switch. **הנתונים בדמו הזה מדומים** (לא מחובר עדיין לחשבון מסחר אמיתי) — המטרה היא להראות כיוון ויזואלי ולקבל פידבק לפני שמתחילים לחבר נתונים אמיתיים.

## איך פותחים
פשוט לפתוח את `gz_dashboard_prototype.html` בכל דפדפן (כפול קליק, או גרירה לחלון דפדפן פתוח).

## מה רואים
- שווי חשבון וכוח קנייה
- פיד חי של קריאות מסחר (MCP) עם סטטוס: עבר / מסומן / נחסם
- ציון אמון התנהגותי של האג'נט (Agent DNA)
- התראות פעילות
- כפתור Kill Switch אינטראקטיבי
- טבלת פוזיציות פתוחות
