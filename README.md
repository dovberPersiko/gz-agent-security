# GZ — Agent Detection & Response

שכבת אבטחה וזהות לאג'נטי AI שסוחרים בשוק ההון. POC עם ברוקר מדומה (לא ברוקר אמיתי — כדי לא לדרוש KYC).

## מבנה הריפו

- `webapp/` — ברוקר מדומה: FastAPI backend + שני עמודים סטטיים, שניהם מוגשים מאותו שירות ב-Render.
  - `/` — אתר מסחר (קנייה/מכירה, פוזיציות, היסטוריית פקודות)
  - `/dashboard` — דשבורד GZ (מעקב חי, Agent DNA, התראות, Kill Switch)
- `mock-broker-mcp/` — שרת MCP דק שקורא לאותו backend, כדי ש-Claude Code (או כל agent) יוכל למשוך/לבצע פעולות על אותו חשבון.

## הרצה מקומית

```
cd webapp && uv run uvicorn main:app --reload --port 8000
```

## מדיניות אבטחה מדומה

הוראת קנייה של 3 מניות ומעלה (מכל מקור — האתר או agent דרך MCP) לא מתבצעת מיידית. היא ממתינה לאישור ידני באתר או בדשבורד. פעולת האישור **אינה** חשופה ככלי MCP, כדי שסוכן לא יוכל לאשר את ההוראה החשודה של עצמו.

## סטטוס

Phase 0 POC: ברוקר מדומה + MCP + דשבורד חי על נתונים אמיתיים (מדומים). עדיין לא קיים: Agent DNA אמיתי, Kill Switch אמיתי, Input Integrity.
